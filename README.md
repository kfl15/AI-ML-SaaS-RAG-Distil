# AI-ML-SaaS-RAG-Distil

Local RAG distillation pipeline for generating grounded instruction datasets
and fine-tuning small, edge-deployable LLMs.

## Key Principles
- Fully local execution (WSL Ubuntu)
- RAG used as teacher
- One-step-at-a-time development
- Strong anti-hallucination discipline
- Git-first workflow

## Environment
- Python 3.11
- Virtual environment located outside repo:
  /mnt/d/venvs/venv311_linux
